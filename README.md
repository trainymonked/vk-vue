# vk-vue

## Ограничения

-   Не работает в Google Chrome в режиме инкогнито (VK OpenAPI)
    -   Почему - VK OpenAPI использует third-party куки, которые в инкогнито по умолчанию не сохраняются
    -   Возможный фикс - переписать на чистые запросы
